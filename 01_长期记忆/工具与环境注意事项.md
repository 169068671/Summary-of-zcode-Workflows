# 工具与环境注意事项

## 网络

- IMA 在开启 VPN 时下载速度可能很慢。进入 IMA 成品下载前，先提醒用户关闭 VPN，由用户操作网络开关。
- 大文件上传或下载中断时，先查任务 ID、本地文件和账单，确认未成功后再重试。

## 密钥与隐私

- API Key 只从已有安全配置或系统钥匙串读取，不写入本仓库、项目文件、日志或对话。
- 记忆库中可以保存"密钥在哪种安全容器中"，不保存密钥值。

## OpenRouter

- 视频生成使用异步任务：提交一次、保存 ID、轮询原任务、成功后立即下载。
- 模型列表、价格、分辨率和时长能力会变化，正式提交前必须实时查询。
