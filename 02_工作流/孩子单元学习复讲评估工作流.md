# 孩子单元学习复讲评估工作流

> [!summary] 核心原则
> 以“孩子能够合上材料，用自己的话准确讲清，并能处理追问和新情境”作为掌握证据。学习、口述、转写、评分、定点纠错、再次复讲和延迟回查必须形成一个闭环。

## 零、流程总览

`确认范围 → 建立知识地图 → 设置固定检查点 → 不计分预检 → 分块学习与即时复述 → 学科专项训练 → 完整复讲 → 证据化评估 → 定点纠错 → 再次复讲 → 7天延迟回查`

### 重新开始规则

- 用户要求“重新开始”时，不对旧一轮重新打分，也不把旧成绩作为本轮起点。
- 旧成绩和旧录音保留为历史证据，不删除、不覆盖。
- 新一轮先做不计分预检，只用来发现已掌握、遗忘、误解和表达不清的内容。
- 从统一教材范围、知识地图和固定检查点重新学习；正式复讲后才产生本轮第一次得分。
- 同一单元后续各轮继续使用相同检查点和量表，保证进步可以比较。

### 最小执行版本

1. 以现行教材、课程标准和学校资料确定单元边界，不混入后续单元。
2. 将单元拆成3至5个学习模块，建立约30个稳定编号检查点。
3. 用5道核心问题进行不计分预检。
4. 每学完一块，合上材料用自己的话复述30至60秒。
5. 完成至少一道本学科的基础、变式或材料应用任务。
6. 合上资料完成8至10分钟完整复讲，家长先听完再追问。
7. 按检查点标记正确、正确但不完整、依赖提示、模糊、错误和未提及。
8. 每轮只修正3至5个最高价值问题，然后重新完整复讲。
9. 连续两次至少90分且无关键错误后通过，约7天后无提示回查。

## 一、何时使用

- 用户要求为孩子整理一个完整学习单元。
- 用户发送孩子的复讲、背诵或口述录音，希望分析掌握情况。
- 需要对同一个孩子的多轮录音进行可比较的跟踪。
- 需要把共享知识、原始证据、个人评估和进度表分开保存。

## 二、每个单元的固定成果

1. 共享的完整单元文档或学习工作流。
2. 稳定编号的知识检查表。
3. 每名孩子的单元跟踪页。
4. 每轮原始录音或压缩副本。
5. 分角色逐字稿。
6. 100分评估报告。
7. 只包含高价值问题的纠错清单。
8. 个人复讲索引和学习进度更新。
9. 连续达标和7天延迟回查记录。
10. 与本批复习资料同步生成的家长行动指南；PDF交付时同时提供可打印PDF，并在Obsidian保留Markdown源稿。

> [!important] 家长行动指南强制伴生规则
> 每次生成单元、阶段、期中、期末或中考复习资料，都必须同步生成针对本批内容的家长行动指南。指南至少包含本轮目标、家长逐步动作、具体提问、最小提示、完成门槛、异常处理、两名孩子分别记录的入口和下一次安排。缺少指南时，不得宣布复习资料完成。

## 三、目录分工

### 共享知识

`10_共享知识/年级/学科/单元/`

保存概念、规则、公式、单位、实验、应用、易错点、口述提纲、问题库和统一评分标准。相同内容只保存一份。

### 学生档案

`20_学生档案/姓名/复讲/学科/单元/`

保存该学生的轮次、录音、逐字稿、得分、错误、提示依赖、下一步和复习日期。

### 年度归档

`99_归档/年份/`

保存旧仓库、历史原音频、处理过程和已经被新版本替代的材料。归档不等于删除。

## 四、建立单元

1. 确认教材、课程标准、学校资料和知识库中的证据边界。
2. 先列完整知识地图，再扩写具体知识点。
3. 将单元分成适合5至10分钟口述的结构。
4. 为知识点建立稳定编号，例如M01—M30；后续轮次保持编号不变。
5. 每个知识点包含：是什么、为什么、条件、公式或事实、例子、反例或易错点。
6. 实验单元必须覆盖目的、器材、步骤、现象、结论、方法、误差方向和安全。
7. 建立关键错误列表；存在关键错误时，无论总分多高都不能通过。
8. 同步编写家长行动指南，把本单元拆成可在35至50分钟内执行的若干次家庭学习，并写明每次的独立产出、家长追问和记录位置。

## 五、孩子首次学习

1. 进行5分钟无分数预检，定位起点；预检不产生正式得分。
2. 把单元拆成3至5块，每块学习后合书复述30秒。
3. 完成至少一道基础题、一道变式题和一个应用或实验任务。
4. 画一页关键词结构图，但正式复讲时不得照读。
5. 孩子能够说出本单元主线后，进入首次正式口述。

### 各学科专项训练映射

| 学科 | 单元训练重点 |
|---|---|
| 语文 | 朗读、意象情感、文本证据、活动任务和表达 |
| 数学 | 概念条件、计算步骤、验算或验根、变式与综合应用 |
| 英语 | 词汇、语法、阅读、听说、口语和写作 |
| 物理 | 公式条件、单位、实验、图像、计算和生活应用 |
| 化学 | 概念、化学用语、实验操作、现象结论和安全 |
| 道德与法治 | 教材观点、材料证据、辨析、规范表达和具体行动 |
| 历史 | 时空定位、史实因果、史料实证、比较和历史解释 |

专项训练随学科调整，但“固定检查点—闭卷复讲—证据评估—定点纠错”的闭环不变。

## 六、正式复讲录音

### 文件名

`YYYY-MM-DD-姓名-单元-第N轮.m4a`

### 默认要求

- 每名孩子单独录制8至10分钟。
- 合上教材、答案和示范稿。
- 孩子先完整讲，家长后追问。
- 家长每次只问一个问题，停顿时等待8至10秒。
- 必须把家长提示留在录音中，不能暂停后补课再继续。
- 给了提示后才回答正确，不能算独立掌握。

## 七、录音接收和处理

1. 确认姓名、单元、轮次和录制日期。
2. 检查音频完整性、格式、时长和解码情况。
3. GitHub普通仓库单文件必须低于100MB，尽量低于50MB；纯语音可压缩为AAC单声道、24kHz、约32kbps。
4. 原始高质量录音应保留在用户原目录或归档区，压缩副本用于知识库和Git。
5. 忠实转写，区分孩子、家长和无法确认的说话人。
6. 听不清处标注，不猜测或修正孩子原话。
7. 语音转文字按[[OpenRouter录音转文字工作流]]执行，只走 OpenRouter 付费大模型，不切换其他识别渠道。

## 八、证据标签

逐个知识编号标记：

- 正确：独立、准确、完整。
- 正确但不完整：主体正确，漏条件、理由或结论。
- 正确但依赖提示：提示后能修正。
- 模糊：无法判断是否真正理解。
- 错误：明确事实、概念、因果或公式错误。
- 未提及：录音没有覆盖，不等于说错，但扣覆盖分。

## 九、100分统一量表

| 类别 | 分值 |
|---|---:|
| 知识覆盖 | 25 |
| 概念准确 | 20 |
| 公式与单位 | 15 |
| 实验理解 | 15 |
| 迁移应用 | 10 |
| 逻辑表达 | 10 |
| 追问回答 | 5 |

对没有公式或实验的学科，应在单元文档中预先说明等价评价项，但同一单元的不同轮次不得临时改变标准。

## 十、评估报告固定顺序

1. 结果和总分。
2. 分类得分。
3. 正确且讲清的内容。
4. 错误、模糊和遗漏，按严重程度排序。
5. 最小纠正提示。
6. 3至7个苏格拉底问题。
7. 应用题，以及适用时的一道实验题。
8. 下一轮录音精确要求。
9. 与上一轮逐项比较。
10. 更新掌握状态、连续90+次数和下次复习日期。

## 十一、定点纠错

1. 每轮只选择3至5个最重要问题，不让孩子重抄整章。
2. 每个错误要求孩子说出“我错在哪里—正确说法—为什么—一个例子或反例”。
3. Zcode一次只给一个苏格拉底问题，从意义、证据、条件变化、反例、比较、应用逐步追问。
4. 给最小提示，不在孩子尝试前直接公布完整答案。
5. 先做3至5分钟定点复述，下一天再做完整复讲。

## 十二、掌握状态

`未开始 → 学习中 → 已录音待评估 → 待纠错 → 待复讲 → 90+第1次 → 已掌握 → 7天待回查 → 保持掌握`

### 通过规则

- 连续两次至少90分。
- 两次都没有关键错误。
- 第二次应换题、换情境或换表达，不能背上一份答案。
- 达标后约7天无提示回查；建议达到85分且无关键错误才标记保持掌握。
- 每完成五个单元，安排一次跨单元解释或应用任务。

## 十三、每轮必须更新的文件

- 个人单元跟踪页：日期、轮次、得分、关键错误、连续90+和状态。
- 复讲索引：新评估文件链接。
- 学习进度：最新得分、状态、连续90+和下次日期。
- 原始录音、逐字稿、评估报告、纠错清单。
- 达标后安排7天回查；未达标只安排定点修复。

## 十四、家长行为规则

- 家长是提问者和观察者，不是录音中的第二位讲解者。
- 不连续补词，不用多个暗示问题把答案喂出来。
- 孩子说错时先记录，录音结束后再教学。
- 关注独立提取能力，而不是本次录音听起来是否流畅。
- 对诚实说“不知道”给予正向反馈；它比自信地说错更容易修正。

## 十五、机械运动样板

完整执行样板位于：

`/Users/wangzirui/孩子们的知识点/10_共享知识/初三/物理/机械运动/机械运动单元完整学习与复习工作流.md`

样板包含：

- M01—M30固定检查点。
- 家长基础、图像、实验和迁移提问卡。
- 大睿第1轮69分后的个性化第2轮要求。
- 小墨第1轮72分后的个性化第2轮要求。
- 音频压缩、转写、评分、状态转换和文件命名规则。

复制到其他单元时，保留闭环、证据标签、量表、状态和通过规则，只替换知识检查表、关键错误、问题卡和个性化任务。

## 十六、验收清单

- [ ] 共享知识和个人证据没有重复混放。
- [ ] 知识编号稳定且覆盖完整。
- [ ] 录音中孩子与家长提示可区分。
- [ ] 评分结论都有逐字稿证据。
- [ ] “未提及”没有被误判成“说错”。
- [ ] 关键错误没有被高总分掩盖。
- [ ] 下一轮任务只有少量高价值修正项。
- [ ] 连续90+、7天回查和跨单元任务均有记录。
- [ ] 音频低于GitHub限制且可正常解码。
- [ ] 复习资料已有针对本批内容的家长行动指南；PDF交付时指南PDF和Markdown源稿均存在。
- [ ] YAML、标签、索引、双链、JSON和敏感信息核验通过。
