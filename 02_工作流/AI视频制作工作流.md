# AI 视频制作工作流

## 标准链路

1. 确认人物主锚点、场景、时长、画幅、分辨率、音频和预算。
2. 锁定连续性：人物、服装、发型、眼镜、道具、空间、机位、光线和色温。
3. 完成脚本、分镜和专业提示词。
4. 先制作并审核主关键帧，通过后才提交视频。
5. 提交前查询模型实时能力、价格、余额和参数。
6. 每个镜头只提交一次，保存任务 ID 并轮询原任务。
7. 成功后立即下载，检查人脸、年龄、表情强度、手部、道具、物理常识、声音和镜头连续性。
8. 只补做问题镜头，不重复已合格的生成。
9. 交付成片、抽帧验收图、提示词、任务 ID 和成本记录。

## 丁美霞项目

开始前必须读取 [[../01_长期记忆/重要人物与数字资产#唯一主锚点（不变）|丁美霞唯一主锚点]] 和 [[../01_长期记忆/丁美霞状态参考体系]]。原 4K 锚点始终负责身份；视频拆帧只负责表情、皮肤、眼神、动作和姿态参考。

- 丁美霞笑容只允许自然微笑，禁止咧嘴笑、露齿大笑、张嘴笑或笑容强度逐渐放大。
- 丁美霞皮肤使用状态参考 S01、S07、S08 辅助校准：紧致健康、肤色自然均匀、眼下清爽、有细腻真实纹理；不得用这些截帧替代主锚点的身份数据。
- OpenRouter + HappyHorse 项目使用 [[OpenRouter HappyHorse丁美霞视频工作流]]。
